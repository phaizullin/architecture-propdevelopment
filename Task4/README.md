# RBAC для Kubernetes в Minikube

```bash

# Шаг 1: Развертывание RBAC системы
./deploy.sh

# Шаг 2: Очистка (при необходимости)
./cleanup.sh
```

## Роли

| Роль | Права | Группы пользователей |
|------|-------|---------------------|
| **security-admin** | Полные права администратора кластера | Специалист по ИБ |
| **devops-engineer** | Управление приложениями и инфраструктурой | DevOps-инженеры, Инженеры по эксплуатации |
| **developer** | Разработка в назначенных namespace | Разработчики |
| **cluster-viewer** | Только чтение ресурсов | Владельцы продукта, Бизнес-аналитики, Менеджеры |
